# QSkill Backend Development Internship Tasks

## Overview

This repository contains backend development projects completed during the QSkill Backend Development Internship. The projects demonstrate REST API development, database management, authentication, authorization, and backend best practices using Node.js, Express.js, and PostgreSQL.

## Projects Included

### 1. Todo Backend

Features:

* Create Todo
* View All Todos
* View Single Todo
* Update Todo
* Delete Todo

Concepts Learned:

* REST APIs
* CRUD Operations
* PostgreSQL Integration
* Express Routing

## Installation

Clone the repository:

```bash
git clone https://github.com/Raktim-03/QSkill-Tasks-Todo.git
```

Install dependencies for each project:

```bash
npm install
```

Create a `.env` file in the respective project directory and configure:

```env
PORT=3000
DATABASE_URL=your_database_url
JWT_SECRET=your_secret_key
```

Run the project:

```bash
npm run dev
```
